# Пример списка на QML
Это список, похожий на список контактов в приложении WhatsApp. Для создания списка использовался язык QML и фреймворк Qt5.
Вы можете пройтись по истории коммитов, чтобы увидеть, как создавалось приложение.


main.qml главный файл в проекте, в котором соединены все компоненты приложения
ContactList.qml содержит прокрученный список контактов
Contact_element.qml содержит модель одного контакта
HelpBar.qml содержит верхнюю строку в приложении с настройками
Icons.qml содержит закруглённые иконки контактов
MainToolBar.qml содержит строку для ввода новых контактов

При нажатии на вкладку "Помощь" в контекстном меню настроек выводится диалоговое окно с подсказками.
